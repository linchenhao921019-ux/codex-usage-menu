# Codex 用量

## 软件用途

Codex 用量用于查看 Codex 剩余额度：

- Mac 端：在 macOS 菜单栏显示 5 小时和 1 周额度剩余情况。
- iOS 端：通过 App 和桌面 Widget 查看同一份额度数据。
